Pooled Mining

What is a Share?

Every hash attempt produces a 256-bit number.
Its value determines what it counts as:

BLK SHARE INVALID — keep hashing
0x0000…0 (hardest) 0xFFFF…F (easiest)

INVALID

Hash above pool target.
Discard, try next nonce.

SHARE

Below pool target.
Submit — proves work.

₿

BLOCK!

Below network target.
Broadcast — earns reward.

Every block is also a valid share. The pool sets an easier target so miners submit shares frequently — proving ongoing work without waiting for a rare block. Share count determines each miner's proportional payout.
